# MKFIFA Rangos GG

Web estatica para analizar jugadores de EA FC 26 usando datos de FUT.GG.

## Actualizaciones

- `npm run update:prices`: actualiza jugadores y precios masivos rapido.
- `npm run update:ranges`: actualiza jugadores, precios y rangos usando cache.

La web publicada se sirve con GitHub Pages desde la rama principal.
